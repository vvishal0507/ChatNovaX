# ChatNovaX 💬

Modern real-time chat application with private messaging, rooms, authentication, and OTP system. 🚀

## 🔥 Features
- Private Chat
- Rooms Chat
- OTP Login
- Online Users
- Profile System

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js
- MongoDB

## 🚀 How to Run
1. Clone repo  
2. npm install  
3. npm start
